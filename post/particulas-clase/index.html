Clase de sistemas de particulas - Unity3D

Clase de sistemas de particulas - Unity3D

Los parametros mas usados son, Base, emision y renderer.

Parametros Base

Duracion: La vida que tendra todas las particulas.

Looping: Crea un loop por el cual crea un loop de las particulas.

Prewarm: Aparecen todas las particulas directamente (No se inician poco a poco).

Start Delay: El tiempo que tardan en empezar.

Start Lifetime: El tiempo desde que sale hasta que desaparece.

Start Speed: La velocidad inicial de las particulas.

3D Start Size: Modifica la malla para estirar las particulas.

Start Size: Tamaño inicial de las particulas.

3D Start Rotacion: Modifica la rotacion de las particulas.

Randomize Rotacion: Rotacion aleatoria.

Start Color: Color inicial.

Gravity Modifier: Gravedad para las particulas.

Simulation Space: Cambiar el pivote.

Scaling Mode: Escalar en el espacio global o loca.

Play On Awake: Reproducir cuando uses Awake.

Max particle: Numero maximo de particulas del sistema.

Auto Random Seed: Donde van a aparecer desde el punto de vision las particulas.

Parametros Emission

Rate: Numero de particulas por segundo que se van a emitir.

Parametros Shape

Shape: La forma que va a coger la particula.

Angle: Angulo de las particulas

Radius: Aumenta o disminuye el radio.

Emit from: Modifica la zona de emision.

Random Direction: Direccion aleatoria.

Parametros Renderer

Render Mode: Billboard (Orientado a la camara), tipo de renderizado Normal Direction: Direccion de la normal

Material: Material a renderizar

Sort Mode: Que particulas iran por delante de lo demas.

Sorting Fugde: Lo mismo

Cast Shadorws: Proyecten sombras

Receiver Shadows: Recibir sombras

Min Particle Size: Tamaño de las particulas minimo.

Max Particle Size: Tamaño de las particulas maximo.

Sorting Layer: La capa

Order in Layer: Orden de la capa.

Billboard alignment: Alineacion del billboard.

Pivot: Pivote general

Light Probes: Si se utilizan los probes

Reflection Probes: Si se utiliza reflection

Anchor override: La posicion
